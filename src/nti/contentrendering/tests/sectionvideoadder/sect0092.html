15.4 Work Backwards

In some math problems, we are given the situation at the end of a process and must figure out what happened to bring about that end. We can often solve these problems by working backwards from the end, reversing each step in the process described in the problem.

All four of these problems have one key feature in common. Each tells a story in which we know the end, and are trying to figure out what the situation was sometime earlier. This is a huge clue to try working backwards from the known end situation.

When we know a lot about the end state of a problem, and want to know where we started, we should try working backwards.
