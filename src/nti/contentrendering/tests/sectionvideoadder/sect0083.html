Solution for Problem 17:

Say you’re one of the players. How many matches will you play?

Each player plays 7 matches, one against each of the other 7 players.

So what’s wrong with the following reasoning?

Each of the eight players plays 7 games, so there are total games played.

Suppose two of the players are Alice and Bob. Among Alice’s 7 matches is a match against Bob. Among Bob’s 7 matches is a match against Alice. When we count the total number of matches as , the match between Alice and Bob is counted twice, once for Alice and once for Bob.

In some problems, we don’t necessarily want to count all pairs, but just some of them.

Five women and four men are at a party.

  1. If each person shakes hands with each other person, then how many total handshakes are there?

  2. If no two men shake hands, but each woman shakes hands with each other person, then how many total handshakes are there?

Solution for Problem 19:

  1. There are 9 people total, and each must shake hands with each other person. So, each person shakes hands 8 times. This gives us a preliminary count of handshakes. But this counts each handshake twice, once for each person in the handshake. So, we must divide by 2 to get a total of handshakes. (Notice that this problem is essentially the same as the round-robin tournament problem, where we’ve replaced “matches” with “handshakes.”)

  2. What’s wrong with this solution:

    Each woman shakes hands with 8 other people, and there are 5 women, for a preliminary total of handshakes. But this counts each handshake twice, once for each person in the handshake. So, we must divide by 2 to get a total of handshakes.

    The problem here is that the count does not count each handshake twice! A handshake between two women is counted twice, but a handshake between a man and a woman is counted only once, just for the woman.

    When dividing by 2 in a counting problem to correct for counting each item twice, make sure you really were counting each item twice!

    We have to handle the handshakes between two women separately from handshakes between a man and a woman. Each of the 5 women shakes hands with 4 other women, for a preliminary count of handshakes. Suppose Alice and Barb are two of the women. Our count of counts the handshake between Alice and Barb twice—once among Alice’s 4 handshakes with other women, and once among Barb’s 4 handshakes with other women. So, here we have to divide by 2, since counts each handshake twice, once for each woman. This gives us handshakes between two women.

    Turning to handshakes between a man and a woman, each of the women shakes hands with 4 men, for a preliminary count of handshakes. Suppose Alice is one of the women and Carl is one of the men. The handshake between Alice and Carl is only counted once, just among Alice’s 4 handshakes with men. So, we don’t divide the preliminary count by 2, because each handshake between a woman and a man is only counted once. There are 20 handshakes between a woman and a man.

    Combining the handshakes between two women with those between a woman and a man, we have handshakes.

  4. A hexagon and its diagonals are shown at the right above. Rather than counting the diagonals directly, we count the diagonals by thinking about each vertex. Each vertex can connect to the other 5 vertices, but 2 of these connections are edges, not diagonals. So each vertex has diagonals. (This is a lot like the handshaking problem! Thinking of a diagonal as a handshake between two vertices, each vertex “shakes hands” with 3 other vertices.)

    There are 6 vertices of a hexagon and each connects to 3 vertices by a diagonal, so it appears that there are diagonals. However, this counts each diagonal twice, once for each vertex at its endpoints. To correct for this, we must divide by 2, giving diagonals.

  5. We can use our process from part (d) for any polygon. A polygon with n sides has n vertices. As in part (c), each vertex can connect to the other vertices, but 2 of these connections are edges, not diagonals. So each vertex has diagonals.

    Since each of the n vertices has diagonals, it appears that there are diagonals. But again, this counts each diagonal twice, once for each vertex at its endpoints. So, we correct our count by dividing by 2, giving diagonals for a polygon with n sides. Checking, we see that this formula does give us the values we found in the first four parts.
