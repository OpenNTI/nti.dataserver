Scenario Generation: Sensitivity Analysis

This module shows how sensitivity analysis can be used to identify vulnerabilities and generate relevant stress scenarios. BIS recommendations are to:

  1. focus on nonlinearities, especially short gamma positions (i.e., non-linear downside due to being short volatility),

  2. conduct scenario analysis of several factors jointly.

We apply sensitivity analysis in an Equity Lending case study (based on research commissioned by the RMA Committee on Securities Lending available here). We first use sensitivity analysis to test the likely downside of a typical money market portfolio. We then analyze how risk changes when equity lending is collateralized with equity instead of cash.
